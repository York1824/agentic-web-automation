# рҹӨ– Agentic Web Automation System - OP-Maps Agent

Ein autonomer KI-Agent, der wie ein menschlicher Spieler **op-maps.com** spielt. Der Agent trifft eigenstГӨndige Entscheidungen, fГјhrt Aktionen aus, lernt durch Feedback und wird Гјber GitHub orchestriert.

## Features

- вң… **Autonome Spieler-Aktionen** (Truppenverschiebung, Angriffe, Verteidigung)
- вң… **Vision-basierte UI-Erkennung** (sieht die Spielkarte wie ein Mensch)
- вң… **Strategisches Denken** (LLM-basierte Entscheidungen)
- вң… **LernfГӨhig** (Feedback durch GitHub Issues)
- вң… **Monitored & Automatisiert** (GitHub Actions Scheduler)
- вң… **Replay & Debugging** (Video-Aufzeichnung aller Aktionen)

## Installation

```bash
# 1. Repository klonen
git clone https://github.com/York1824/agentic-web-automation.git
cd agentic-web-automation

# 2. Dependencies installieren
npm install

# 3. Environment-Variablen setzen
cp .env.example .env
# Bearbeite .env mit deinen Credentials
```

## Quick Start

```bash
npm run start:debug
```

## Feedback geben (Trainieren)

GitHub Issue erstellen mit Tag `[AGENT-FEEDBACK]` um den Agent zu trainieren!
